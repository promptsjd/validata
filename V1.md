# Validata — Version 1.0.0

## What's in V1

### Core capture
- Automatic post capture as you browse Reddit during a session
- Click-to-capture individual comments with hover highlighting
- Captures post title, body, URL, timestamp, and comment text
- Parent-child node relationship preserved (post → comment tree reconstructable from exports)

### Projects
- Create named projects with independent capture settings
- Per-project toggles: timestamp, post link, comment capture
- Settings are locked while a session is active to prevent mid-session changes
- Default settings — save your preferred options once; every new project inherits them

### Sessions
- Session tied to the active Reddit tab
- Live capture counter in the side panel
- Session survives panel close; resumes on reopen

### Export
- **CSV** — flat node list, spreadsheet-ready (8 columns: id, projectId, type, parentId, postTitle, text, url, timestampISO)
- **JSON** — full fidelity including project metadata (name, createdAt, schemaConfig, counts) and all nodes
- Clear data or delete project from the context menu

### Architecture
- Chrome Manifest V3 (service worker background)
- Side panel UI (no popup)
- All storage in `chrome.storage.local` — fully offline
- Clean separation: storage service / message router / UI layer / content scripts

---

## Known Limitations

- Reddit only (by design)
- No search across captured nodes
- No tagging or annotation on individual nodes
- Single active session at a time
